WordPress Installation

  1. Log-in to your WordPress site
  2. Ensure the Visual Composer plugin is activated.
  3. Navigate to the Plugins Menu > Click Add New > Click the Upload link
  4. Click Choose File > Locate the "ultimate-pricing-tables" .zip file > Click Install Now
  5. Click Activate Plugin

Add a Pricing Table to a Page / Post

1. Create / Edit a Page or Post > Click Add Element and search for "Pricing Table".

 

2. Edit the Layout of your Pricing Table.

1. Select a Column Layout ( Recommend to do this first before editing each of the Plans ).
2. Edit each Pricing Plan ( Color, Icons, Font Colors etc ) by clicking the Edit icon.
3. Drag & Drop the individual Pricing Plans by dragging the title.
4. Edit the Pricing Table options ( Animation, Padding etc ).

 

3. Edit individual Pricing Plans ( General Tab ).

1. Title : This is the title of the Pricing Plan.
2. Featured Plan : Check this option to set this Plan as the Featured Plan, it will offset the Plan from the rest of them.
3. Price : If your Plan requires a price, set it here.
4. Currency : If you wish to add a currency to your "Price", add one here .e.g $.
5. Per : If you require a Per value, enter it here e.g. / mo , / day, / yr etc.
6. Content : Enter the content of your Plan into this section, it's recommended to use Lists.
7. Button Text : If you wish to add a Button, enter text here for the it.
8. Button Link : Add a URL to link the Button to.
9. Button Target : Set the target of the Button link.
10. Icon : If you wish to add an Icon to the Plan, add a Font Awesome Icon class to this field e.g. fa-compass.

 

4. Edit individual Pricing Plans ( Design Tab ).

1. Plan Color : Select a Plan Color, choose from default options or create a Custom Color ( See Creating a Custom Plan ).
2. Border Style : Select a Plan border style, or set to None to disable the border completely.
3. Shadow Effect : Add a Shadow Effect to your Plan.
4. Icon Color : If you have set an Icon ( General tab ), you can choose the Color with this option.
5. Icon Background Color : If you have set an Icon ( General tab ), you can choose the Background Color with this option.

Create a Custom Plan

  1. Edit a Pricing Plan




  2. Select the Design Options and select Custom Color from the Plan Color select menu.


1. Plan Color : To create a Custom Plan, select Custom Color from the Plan Color select menu.
2. Custom Title Background Color : Set the title Background Color.
3. Custom Title Color : Set the title text Color.
4. Custom Background Color : Set the Background Color of the entire Plan.
5. Custom Text Color : Set the main Text Color of the Plan.
6. Custom Border Color : If you have a Border Style set, you can set a Border Color with this option.
7. Custom Button Color : If you have a Button enabled, you can set the Color of the Button with this option.
8. Custom Button Text Color : If you have a Button enabled, you can set the Color of the Button Text with this option.

Import Demo Content

To import the demo content as seen on the demo site , please see the following instructions.

  1. Log-in to your WordPress admin - Goto Tools → Import → Select WordPress.
  2. If required, download and active the WordPresss Importer plugin. Once activated you may need to repeat steps above.
  3. Locate and select the ultimate-pricing-tables/xml/demo.xml → click the Upload File and Import button.
  4. Assign admin user to an existing user. e.g. admin → select Download and Import file attachments.
