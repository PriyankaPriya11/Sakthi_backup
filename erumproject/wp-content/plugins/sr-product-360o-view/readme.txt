== Description ==

Now present your product to your customers with extra detail and in a more engaging way.
SR Product 360&#176; view provides you with the capability to add  360&#176; image rotation to your store products.
No more boring images slide show when you can give a complete 3D view of your products.

== Installation ==
1. Install & active WooCommerce plugin, if not activated already.
2. Upload the entire sr-product-360o-view folder to the /wp-content/plugins/ directory.
3. Activate the plugin through the Plugins menu in WordPress.
